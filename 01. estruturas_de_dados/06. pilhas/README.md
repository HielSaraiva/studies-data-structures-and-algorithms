## Pilha

É considerada do tipo **LIFO** (Last In - First Out), ou seja, o último elemento inserido será o primeiro a ser removido. Nessa estrutura, cada elemento armazena um ou vários dados e um ponteiro para o próximo elemento, permitindo o encadeamento e mantendo a estrutura linear. Qualquer estrutura desse tipo possui um ponteiro denominado TOPO, no qual todas as operações de inserção e remoção acontecem. Assim, as operações ocorrem sempre na mesma extremidade da estrutura.
