# Implementar essas operações em uma Lista

* INSERIR
* ALTERAR
* EXCLUIR
* BUSCAR
* ORDENAR
* LISTAR-TODOS

## Dados da questão

### Versão 1

   ``````java
   #define capacidade 10

   struct lista
   {
      int tamanho;
      int cap;
      int vetor[capacidade];
   };

   void iniciar(struct lista *p)
   {
      p->tamanho = 0;
      p->cap = capacidade;
   }
   ``````

##

### Versão 2

   ``````java
   #define capacidade 10

   struct lista
   {
      int tamanho;
      int cap;
      int *valores;
   };

   void iniciar(struct lista *p)
   {
      p->tamanho = 0;
      p->cap = capacidade;
      p->valores = malloc(capacidade * sizeof(int));
   }
   ``````
